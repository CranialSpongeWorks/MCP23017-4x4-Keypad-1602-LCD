# MCP23017-4x4-Keypad-1602-LCD
MCP23017 4x4 Keypad 1602 LCD


This sketch runs on a NANO and uses the MCP23017 I2C I/O Expander to interface a 4x4 matrix keypad and an 1602 LCD module.
